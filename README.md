## Intro

This is a hobby project I'm making to harden my Rust skills.

The goal of this repo is to setup an on-chain order book on Solana, and providing an API backend (also in Rust) to interface with the ecosystem to provide a gasless UX experience. Once this is done, stage 2 will exhance the backend to become a Relayer and Order Engine, only using the smart contracts to execute settlements.
